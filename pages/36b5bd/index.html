# Vue 概述

# 1、Vue.js 特点

  • 简洁：HTML 模板 + JSON 数据，再创建一个 Vue 实例，就这么简单。
  • 数据驱动：自动追踪依赖的模板表达式和计算属性。
  • 组件化：用解耦、可复用的组件来构造界面。
  • 轻量：~24kb min+gzip，无依赖。
  • 快速：精确有效的异步批量 DOM 更新。
  • 模块友好：通过 NPM 或 Bower 安装，无缝融入你的工作流。

# 2、Vue.js 安装

  • （1）独立版本

直接下载并用 <script> 标签引入，Vue 会被注册为一个全局变量。

  • 开发版本：包含完整的警告和调试模式
  • 生产版本：删除了警告，进行了压缩

  • （2）使用 CDN 方法

  • （3）NPM 方法

  • 由于 npm 安装速度慢，可使用淘宝的镜像及其命令 cnpm，安装使用介绍参照：使用淘宝 NPM 镜像 (opens new window)。

  • npm 版本需要大于 3.0，如果低于此版本需要升级它。

  • 在用 Vue.js 构建大型应用时推荐使用 NPM 安装：

npm install vue
