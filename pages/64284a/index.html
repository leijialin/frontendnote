# Windows 软件卸载清理工具

# Geek Uninstaller

Geek Uninstaller 非常简洁轻巧，只有 6M 大小，绿色软件，解压即用。支持 Windows 商店 UWP 应用的卸载，查看软件注册表和安装目录等。

  • Geek Uninstaller-官网 (opens new window)
  • Geek Uninstaller-绿色版 (opens new window)

# Revo Uninstaller

  • 官网 (opens new window)
  • 绿色版 (opens new window)

集软件卸载和系统垃圾清理于一体的万能工具箱，支持多种卸载模式选择和软件追踪安装。

卸载时可以自动创建还原点和备份，软件包含垃圾文件、Office、浏览器、系统清理功能和 Windows 工具等。功能十分全面。

# Your Uninstaller

  • 官网 (opens new window)

Your Uninstaller! 也是一款专业的软件卸载工具，软件非常简洁，虽然很多年没有更新了，但是依然很好用！卸载时可以视情况选择四种卸载模式，附带了六款实用小工具。
