SVN使用

# VisualSVN Server 常见问题

1、更改SVN版本库存放路径

  • 打开VisualSVN Server，在VisualSVN Server（Local）上右键点击：Propeties，打开属性对话框。
  • 在Storage 页中，有默认的Repositories Root地址，点击Browse... ，选择你新的版本库存放地址即可。

# SVN 常见问题

1、SVN项目地址改变后如何切换到新的地址

  • 右键项目文件夹 TortoiseSVN => Relocate 在 To URL 中输入新的仓库地址点击 OK 然后填入账户名密码即可！

  • Linux下

# 查看当前仓库地址
svn info

svn switch --relocate (Old Repository Root) (New Repository Root)
# 示例：svn switch --relocate https://127.0.0.1:8443/svn/items https://123.34.56.78:8443/svn/items
